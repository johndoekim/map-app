# Taroot(따룻)
빅데이터와 머신러닝을 사용한 따릉이 기반 사용자 친화 추천 경로 서비스.

단순 경로 추천뿐만이 아닌, 사용자들간 개인화 된 경로를 커뮤니티에서 추천과 동시에 소통 할 수 있는 기능 제공.
